# Demo checklist

Record a short screen capture or prepare screenshots that show:

1. `terraform init` and `terraform apply` completed successfully
2. the API Gateway base URL output from Terraform
3. the frontend running with `npm run dev`
4. raw meeting notes pasted into the textarea
5. the returned summary, key decisions, and action items
6. the S3 bucket containing a new JSON log object
7. the repository structure and README
